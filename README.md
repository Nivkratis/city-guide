**Сонюшкин Никита**<br>
**Spring / вариант задания 2**<br>
Путеводитель по городу<br>
Веб-приложение и REST API для поиска достопримечательностей города, управления местами и работы с отзывами пользователей<br>

**Технологический стек**<br>
Backend:Java 22, Spring Boot 4.1.0 (Spring Web, Spring Data JPA, Lombok)<br>
Database: PostgreSQL + скрипт развертывания БД через Flyway<br>
Frontend: HTML5, CSS3, JavaScript с Bootstrap 5<br>
Test: JUnit 5 + Mockito<br>
Используется Docker в качестве инструмента контейнеризации<br>
Пароли пользователей хешируются <br>

**Функциональные возможности**<br>
Пользователи: регистрация и аутентификация с валидацией заполнения полей на фронтенде<br>
Достопримечательности: добавление новых мест (название, категория, точные географические координаты), отображение общего списка<br>
ГИС-поиск: поиск мест в заданном радиусе (в км) от указанных координат с фильтрацией по категориям и минимальному среднему рейтингу<br>
Отзывы и оценки: возможность выставить оценку (от 1 до 5) к любому месту. Написание текстового комментария является необязательным<br>

**Развертывание и запуск проекта**<br>
Так как всё работает через Docker, то можно при желании обойтись только им. При установленном на ПК Docker,
собираем проект командой ./mvnw clean package -DskipTests в консоли. Затем docker-compose up --build -d. И 
приложение будет доступно по адресу http://localhost:8080. Либо собирать и запускать проект локально, как обычно.
